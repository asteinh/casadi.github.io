Algorithmic Differentation (AD)

CasADi's backbone is a symbolic framework implementing forward and reverse mode of AD on expression graphs to construct large-and-sparse Jacobians and Hessians. These expression graphs, encapsulated in Function objects, can be evaluated in a virtual machine or be exported to stand-alone C code.
{{< highlight octave >}} import casadi.* % Create scalar/matrix symbols x = MX.sym('x',5); % Compose into expressions y = norm(x,2); % Sensitivity of expression -> new expression grad_y = gradient(y,x) % Create a Function to evaluate expression f = Function('f',{x},{grad_y}); % Evaluate numerically grad_y_num = f([1;2;3;4;5]) {{< /highlight >}} {{< highlight python >}} .... {{< /highlight >}} {{< highlight cpp >}} 000 {{< /highlight >}}
